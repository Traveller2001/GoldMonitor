# GoldMonitor

实时黄金价格桌面悬浮窗，数据来自招商银行（上海金交所 Au(T+D)）。

![Python](https://img.shields.io/badge/Python-3.9+-blue) ![PyQt6](https://img.shields.io/badge/PyQt6-GUI-green) ![License](https://img.shields.io/badge/License-MIT-yellow)

## 功能

- 半透明圆角悬浮窗，始终置顶显示
- 实时显示 Au(T+D) 人民币/克价格
- 涨跌幅变色提醒（红涨绿跌）
- 今日涨跌幅百分比显示
- 今日最高/最低价显示
- 可配置区间时长，对比区间内价格变化
- 支持设置刷新间隔、变色阈值
- 价格突破阈值时发送系统通知
- 最近日志查看器（自动保留近 1 小时记录）
- 毛玻璃风格设置面板与日志面板
- 系统托盘图标，支持显示/隐藏
- 鼠标拖拽移动窗口位置
- 跨平台支持（macOS / Windows / Linux）

## 快速开始

```bash
pip install -r requirements.txt
python main.py
```

## 使用方法

- **拖拽移动**：左键按住窗口拖动
- **右键菜单**：设置 / 最近日志 / 刷新 / 退出
- **系统托盘**：右键可显示/隐藏窗口、打开设置、查看日志

## 设置项

| 设置 | 默认值 | 说明 |
|------|--------|------|
| 刷新间隔 | 30 秒 | 价格更新频率 |
| 变色阈值 | 0.5% | 相邻刷新涨跌超过此值变色 |
| 区间时长 | 5 分钟 | 对比该时段内的价格变化 |
| 高价通知 | 0（关闭） | 金价 ≥ 此值时发送通知 |
| 低价通知 | 0（关闭） | 金价 ≤ 此值时发送通知 |

## 数据来源

招商银行黄金行情 API（上海金交所实时报价）。

## 依赖

- Python 3.9+
- PyQt6
- requests
